## 주요 기능

### ⭐️ 자기소개서 분석하기
- 자소서에서 핵심 키워드 추출(예: 팀워크, 협업)
- 직무 연관성에 대한 평가 제공

### ⭐️ 내 자소서는 몇 점?
- 구체적인 문장 피드백 제공 (예: 구체적인 경험을 추가하세요.)​
- 문장 가독성 점수 출력

### ⭐️ AI가 교정해주는 내 자소서​
- 맞춤법 및 문장 교정​
- 예문 제시 및 문장 추천
